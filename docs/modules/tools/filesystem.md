# Filesystem Tool

Provides file system operations for agents.

## Module ID

`tool-filesystem`

## Capabilities

- Read files
- Write files
- List directories
- File metadata

## Configuration

| Option | Type | Description |
|--------|------|-------------|
| `allowed_paths` | list | Allowed directory paths |
| `read_only` | bool | Disable write operations |
