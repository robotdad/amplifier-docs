# CLI API

The `amplifier-app-cli` package provides the reference CLI implementation.

## Main Entry Point

::: amplifier_app_cli.main
    options:
      show_root_heading: true
      members:

## Commands

CLI commands are implemented in `amplifier_app_cli.commands`:

- `run` - Execute prompts
- `session` - Session management
- `profile` - Profile management
- `provider` - Provider management
- `module` - Module management
- `collection` - Collection management

See [CLI Reference](../../user_guide/cli.md) for usage documentation.
